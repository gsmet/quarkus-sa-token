== Installation

If you want to use this extension, you need to add the `io.quarkiverse.satoken:quarkus-sa-token-sso` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.satoken</groupId>
    <artifactId>quarkus-sa-token-sso</artifactId>
</dependency>
----
